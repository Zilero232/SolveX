## About

**Chatovo** is a real-time communication app where conversations happen in *rooms*, not in endless chat threads. Spin up a room in a single click, share the link with friends, and start talking — voice and video work out of the box, with zero accounts to wrangle on their side beyond a quick sign-in.

The app is built around three ideas:

- **Rooms over channels.** Each room is its own little world — public, private, or password-protected. No noisy server lists, no nested categories. Just rooms.
- **One experience, everywhere.** Open Chatovo in any modern browser, or install the native desktop build. The interface, the rooms, the accounts — all the same.
- **Fast, quiet, focused.** Designed to fade into the background while you talk. Crisp audio, low-latency video, and an interface that gets out of the way.

## Highlights

- **Instant voice & video rooms** with low-latency WebRTC and a modern SFU backbone.
- **Private rooms** protected by a password — only those who know it can join.
- **One-click sign-in** via Google, or a classic email + password account.
- **Lightweight desktop app** that ships as a native binary, not a bundled browser.
- **A single, consistent UI** across web and desktop — designed to feel calm rather than crowded.
- **Self-hostable.** The whole thing runs on your own server with a single Docker Compose file and an automatically issued TLS certificate.

## Who it's for

Chatovo is for small groups, study circles, dev teams, gaming friends, and anyone who wants a private space to talk without ads, trackers, or noisy social features pushed on top.

If you've ever wished Discord were simpler, quieter, and yours — Chatovo is that.
